pathomechanism:
- [[why typhoid enters via small intestine ie ileum and not others]]
- cross the epithelia, gets inside macrophages w/o killed (Trojanization)
- via Trojan macrophages get into lymphoid hubs ([[RES]]/[[MPS]]), inside there recruit other macrophages, cause enlargement e.g. hepatosplenomegaly
- from there get back to systemic circulation as patrolling Trojan macrophages
- if lysed (has replicated enough inside macrophages?), typhi gets out free,
	- if dies (e.g. lacking macrophageal sustenance), release [[LPS]] (lipoprotein A is hidden before), trigger systemic inflammation response
	- if alive, may go around; weaken endothelia, get localized into organs, either free living (esp e.g. gallbladder) or inside macrophages
