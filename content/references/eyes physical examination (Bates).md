# [[eye lacrimal apparatus]]
- inspect [[edema]] in [[lacrimal gland]] and [[lacrimal sac]]
- excessive tearing?
	- <- increased production <- [[conjunctivitis]] or [[cornea]]l irritation
	- <- impaired drainage <- [[ectropion]] and/or [[nasolacrimal duct obstruction]]
- excessive dryness?
	- <- impaired secretion <- [[Sjögren syndrome]]

# [[conjunctiva]] and [[sclera]]
- depress lower lid, ask to look up -> check [[sclera]] and [[conjunctiva]]. inspect:
	- color
	- vascular pattern: slight vascularity is normal
	- nodule or swelling [[chemosis]]
- if need fuller view, spread the eyelid on cheek bone and [[eyebrows]], ask to look each side and down -> check [[sclera]] and [[bulbar conjunctiva]]
- lid eversion for (upper) [[palpebral conjunctiva]] (though at some places e.g. Yakes Telkom, 'unethical')

# [[cornea]] and [[lens]]
- oblique lighting, inspect opacities:
	- in [[cornea]]
	- in [[lens]] which may be visible through [[pupil]]

# [[iris]]
- horizontal light from temporal, check for crescentic shadow on medial side (see: [[shadow test (iris)]])

# [[pupil]]s
- inspect
	- size
	- shape
	- symmetry
		- [[simple (benign) anisocoria]], dd with:
			- [[Horner syndrome]]
			- [[oculomotor nerve paralysis]]
			- [[tonic pupil]]
